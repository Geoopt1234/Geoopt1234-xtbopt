## Software Requirements
### 1. xtb 6.7.1pre (5071a88)
**Download from the official source:**
- **Windows:** Download the `xtb-6.7.1pre` Windows binary from the [grimme-lab/xtb 6.7.1](https://github.com/grimme-lab/xtb/releases/tag/v6.7.1).
- **Linux/macOS:** Follow the compilation instructions on the official page.
**Verification:** All calculations in this database were performed with in xtb 6.7.1pre (5071a88).
To check versions, you can always use `xtb --version`. It should look like this
```
      -----------------------------------------------------------
     |                   =====================                   |
     |                           x T B                           |
     |                   =====================                   |
     |                         S. Grimme                         |
     |          Mulliken Center for Theoretical Chemistry        |
     |                    University of Bonn                     |
      -----------------------------------------------------------

   * xtb version 6.7.1pre (5071a88) compiled by 'Marcel@Raven' on 2024-07-23

normal termination of xtb
```
### 2. InChI (Version 1.07.4)
**Download from the IUPAC/NIST InChI Trust:**
- **All Platforms:** Download the stable version `1.07.4` from [www.inchi-trust.org/downloads](https://github.com/IUPAC-InChI/InChI/releases/tag/v1.07.4).
